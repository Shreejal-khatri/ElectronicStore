ElectronicStore

MERN Stack Web Application

1. Introduction

ElectronicStore is a web-based e-commerce application developed using the MERN stack. The system is designed to simulate an online electronics store where users can view products, manage a shopping cart, and place orders. The project demonstrates full-stack web development using modern JavaScript technologies.

2. Technology Stack

The application is developed using the following technologies:

MongoDB – Database

Express.js – Backend framework

React.js – Frontend library

Node.js – Server-side runtime

HTML, CSS, JavaScript

RESTful APIs

3. System Features
3.1 User Features

View available electronic products

View detailed product information

Add products to cart

Update or remove products from cart

Place orders

3.2 Backend Features

User authentication

Product management (Create, Read, Update, Delete)

Cart management

Order processing

API-based communication between client and server
